Sheetbroker FAQs

Take a look at the frequently asked questions for sheetbroker.

Q.  What if you create the wrong team enrollment?For example: I created an enrollment that linked Team 5’s sheet to Team 10.
A. Unplug the micro:bit sender. Delete the enrollment. Make a new enrollment with the correct team number and google sheet. Check the micro:bit sender code is the correct team number. You may have to clear the sheet if it already has data. Plug in the micro:bit sender.

Q.  My enrollment is correct but what if I (or a student) programmed the micro:bit sender code with the wrong team number? For example: Team 5’s sheet is connected to Team 5, but a student in Team 5 programmed their sender micro:bit to be team 15. 
A. Unplug the micro:bit sender. Correct the team number on the sender code and re-download it onto the sender micro:bit. Leave the enrollment as it is. Note: if you already had a team setup for the incorrect sender team number (15 in the above example) there might be data that needs to be deleted in that team’s sheet (team 5’s sheet in the above example).

Q.  Can I send data from one micro:bit to multiple sheets? For example: I want one micro:bit to send data to Team 5 and Team 15’s sheets.
A. This is not currently possible. Sheetbroker can only connect a micro:bit to a single sheet. But you can connect multiple micro:bits to the same sheet, they just need to use the same team number.

Q.  Can I have multiple micro:bit senders sending data to the same Google spreadsheet? For example: Team 4 wants to have micro:bit A to send temperature data and micro:bit B to send light data.
A. Yes. Just be sure all the micro:bit senders are all using the same team number. You can see how many micro:bits are connected to a single sheet by looking at the enrollments page.

Q.  Can I change the team number that a micro:bit is sending?
A.  Yes. Just change the sender code on that micro:bit. Be sure to either make an enrollment for that new team number or delete the old team number enrollment if you are no longer using it. 

Q.  I have a sheet connected to the correct team, a receiver micro:bit plugged into the pi, a sender micro:bit sending with the proper team number, and I see hearts/faces flashing on both micro:bits, but I don’t see data in the sheet?
A. Check that you have shared the google sheet with Everyone. Give sheetbroker a few minutes after that change and if you are still having issues, contact Felix (Sheetbroker support personnel).
