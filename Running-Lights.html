Running Lights

Beginner | MakeCode, Python | Neopixels, Buttons | Shift, Pixel color

Step 1: Make it

What is it?
LEDs appear to be moving at different rates

How it works?

Like ants crawling along the ground, a light appears to be crawling along the strip. 

This program shifts the light that is on to the next pixel, clearing the previous pixel, giving the illusion of the light moving.

Button A decreases the rate of the lights “moving” and Button B increases the rate. This is done by changing the pause time between lights being shifted.

What you need?
Micro:bit (or Makecode simulator)
MakeCode or Python editor
Battery pack (optional)
Neopixels (optional)

Step 2: Code it

Download HEX Open in MakeCode

Step 3: Improve it

Make it look like two lights are moving along the strip
Make an off light move while all the other lights are on
