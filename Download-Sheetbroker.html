Download Sheetbroker Software

Decide which version of Sheetbroker that you want and copy Sheetbroker to an SD card.

Step 1: Choose a Sheetbroker version

Router Version 
Pick the router version if you are using sheetbroker in a setting, such as a school, that uses wifi networks that require you to log in through a captive portal. 

What you need when using the router?
microSD card of size 16GB or greater
Raspberry Pi Model 3 (other models are untested)
A computer or phone
A GL.iNet mini smart router

OR

Raspiwifi Version 
Pick the Raspiwifi version if you are using sheetbroker at home on a regular home wifi network. 

What you need when using Raspiwifi?
microSD card of size 16 or greater
Raspberry Pi Model 3 (other models are untested but must have a wireless chip)
A computer or phone

Step 2: Download Sheetbroker software

Raspiwifi Version Router Version

OR

Do not unzip the file when it is done downloading. Note: It may take a while for the file to download because of the large size.

Step 3: Copy Sheetbroker software to an SD card

Transfer the downloaded Sheetbroker software onto a microSD card that is at least 16 GB. Even though the software may fit on an 8GB card, there will not be enough memory left to run the application.

If you are unfamilar with flashing images to a microSD card, we suggest using Raspberry Pi Imager. This is a utility application that will allow you to copy the software to the mircoSD card. After downloading and openting Raspberry Pi Imager, click "Choose OS". Then select "Use Custom" from the bottom of the menue. Then select the Sheetbroker software that you downloaded in Step 2.

Note: depending on your computer, you may need an SD adapter in order to copy the software onto the microSD card.

Step 4: Insert the microSD card into the Raspberry Pi

The microSD card slot on the Raspberry Pi 3 is located on the side opposite the USB ports and on the bottom of the Pi. Insert the microSD card with the writing on the card facing away from the board.

Troubleshooting Tips

  • If the SD card is not showing up when you try to write the image using Raspbery Pi Imager, make sure that the SD card adapter is unlocked. Some SD card adapters have physical locks (a little slider on the side of the adapter) that will prevent them from being recognized by the computer.
  • If you need to remove an old image or other information from an SD card and deleting the files isn’t working, on a Mac you can use disk utility to erase and clear the card.
