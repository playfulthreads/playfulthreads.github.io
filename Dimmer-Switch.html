Dimmer Switch

Beginner | MakeCode, Python | Neopixels, Buttons | Brightness

Step 1: Make it

What is it?
The buttons make the LEDs bright and dim

How it works?
This program acts like a two level dimmer switch, one button makes the LEDs bright and the other makes the LEDs dim.

The program uses the set brightness functionality to brighten or dim the lights depending on if Button A or B is pressed.


What you need?
Micro:bit (or Makecode simulator)
MakeCode or Python editor
Battery pack (optional)
Neopixels (optional)

Note: the simulator does not show changes in brightness very well

Step 2: Code it

Download HEX Open in MakeCode

Step 3: Improve it

.Change the brightness levels to explore the range of brightness possible
Add another level of brightness with an A+B press
