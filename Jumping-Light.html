Jumping Light

Beginner | MakeCode, Python | Neopixels, Buttons | Clear, Pixel color

Step 1: Make it

What is it?
Each button press turns on a random light

How it works?

The classic whack-a-mole game has a mole randomly appear in different holes across a board. This program recreates this game by having a single light appear in a different location with each button press.

The program clears all previous lights and then lights up an individual pixel (LED) with each button press.

What you need?
Micro:bit (or Makecode simulator)
MakeCode or Python editor
Battery pack (optional)
Neopixels (optional)

Step 2: Code it

Download HEX Open in MakeCode

Step 3: Improve it

Make the lights “move” as you shake the micro:bit

To be more like whack-a-mole make the light appear in random places constantly (rather than on a button press)
